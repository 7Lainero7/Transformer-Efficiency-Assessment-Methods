# Данные исследования: Сравнение LLM для генерации текста

## Описание
Датасет содержит результаты экспериментального сравнения 14 языковых моделей в задаче генерации структурированных текстов на русском языке.

## Структура
- 14 моделей (web и local)
- 5 бизнес-сценариев
- 70 сгенерированных текстов
- Оценки 5 экспертов (3 человека + 2 LLM)
