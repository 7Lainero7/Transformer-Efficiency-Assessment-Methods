# Данные исследования: Сравнение LLM для генерации текста

## Описание
Датасет содержит результаты экспериментального сравнения 14 языковых моделей в задаче генерации структурированных текстов на русском языке.

## Структура
- 14 моделей (web и local)
- 5 бизнес-сценариев
- 70 сгенерированных текстов
- Оценки 5 экспертов (3 человека + 2 LLM)

## 📓 Jupyter Notebook

В репозитории `model_testing.ipynb` - интерактивный блокнот для тестирования языковых моделей.

### Возможности notebook:
- Загрузка моделей с Hugging Face
- Генерация текстов по заданным промптам
- Сохранение результатов в JSON и HTML
- Автоматический анализ качества сообщений

### Как использовать:
1. Откройте в Google Colab или Jupyter
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите все ячейки
4. Выберите модель для тестирования из списка

### Требования:
- Python 3.8+
- CUDA-совместимый GPU (рекомендуется)
- Токен Hugging Face (бесплатно)
